## kadiripek.com

Kişisel portfolyo sitem. Eğitim, iş deneyimleri, projeler ve sertifikalarımı içeriyor.

## Teknolojiler

- PHP / Laravel 13
- MySQL
- Tailwind CSS
- Vite

## Kurulum
```bash
git clone https://github.com/kadiripek/kadiripek.git
cd kadiripek
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
